Bem-vindo ao TaskMaster

Este sistema foi desenvolvido para centralizar, organizar e priorizar suas tarefas diárias de forma simples e eficiente.

Utilize o menu lateral para navegar. Cadastre-se ou faça login para liberar as funcionalidades completas do CRUD.

Recursos do Sistema:

  • Cadastro de tarefas com níveis de prioridade.
  • Filtro de busca em tempo real.
  • Armazenamento local persistente (LocalStorage).
