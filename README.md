# TaskFlow – Controle de Tarefas

Sistema web de gerenciamento de tarefas desenvolvido com HTML, CSS, JavaScript e LocalStorage.

## 📋 Sobre o Projeto

TaskFlow é um sistema CRUD completo para controle de tarefas, com autenticação de usuários, interface moderna e armazenamento local via LocalStorage.

## ✅ Funcionalidades

- **Login / Cadastro de usuário** com sessão persistente
- **Cadastro de tarefas** com campos: Tarefa, Prioridade, Status, Descrição (opcional) e Anexo (opcional)
- **Listagem em tabela** com todos os dados
- **Edição inline** diretamente na tabela
- **Exclusão** com confirmação em modal
- **Busca e filtros** por nome, prioridade e status
- **Validação** de campos obrigatórios e nomes duplicados
- **Tela Home** com estatísticas e descrição do sistema
- **Menu lateral** com navegação entre telas
- **Botão Sair** para deslogar

## 🛠️ Tecnologias

- HTML5
- CSS3 (custom properties, flexbox, grid)
- JavaScript (ES6+)
- LocalStorage (armazenamento persistente no navegador)
- Font Awesome (ícones)
- Google Fonts – Inter
